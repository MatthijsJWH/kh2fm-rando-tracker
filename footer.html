Kingdom Hearts 2 Final Mix Item Tracker - Version 1.4.2
Click here to use the pre-1.4 version

This is a tracker intended for use with the Kingdom Hearts 2 Final Mix Randomizer by Valaxor, for the Garden of Assemblage (GoA) Mod by Sonicshadowsilver2.

It allows the tracking of progress such as world completion, drive forms, magic, etc. For a more detailed explanation, see Help.

It was initially developed by Kokemon198, however has since entirely been rewritten by Zax with an entirely different codebase. Massive thanks to Televo for providing really nice images that replaced all of the previous ones, and adding new items.

The GitHub repository with the original code for this site can be found here.

If you have any questions/issues, you can find me in the KH2FM Rando Discord server, Twitter (@Z_ax__), or GitHub. You can also find Televo in the rando Discord for questions regarding their additions.

Left Click

The item will become activated, lighting up to full opacity.

When clicking on a world more than once, if applicable, a number will appear in the bottom right indicating how many visits have been completed. Drive, magic, and summons have similar systems for levels.

After all visits have been completed, clicking again will either trigger a symbol to appear in the top left, for defeat of the Data Org/Absent Sillhouette fight, or reset the cell back to its original state.

Note: 'Visits' for Atlantica/100 Acre Wood represent not whole world completion but segment completion. That is, song completion for Atlantica (including the tutorial), and pages for 100 Acre Wood.

Note: The counter for Sora's Level will increase in increments of +1 from LV1-15, then +5 from LV15-50 and +10 from LV50-99.

Middle Click

A red cross will be put over the item, intended to be used if a world/item is disabled in the seed. (Items can now be completely removed through settings.)

This hasn't been rigorously tested yet. Use at your own risk.

Lower Atlantica/100 Acre Wood Total

By default the Atlantica tutorial and Pooh's House are included in the total. This removes them, reducing the total from 6 to 5.

Disable image shadows

All images in the tracker are displayed with a slight drop shadow. Disabling these can significantly improve performance on slower devices.

Icon order

These settings for order and removing will be changed to be more convenient in the future. These are basic temporary settings. The input to these fields should be a series of indices. E.g. 4 7 1 corresponds to 4th icon, 7th icon, 1st icon.

Remove icons

This should only be used if a custom order isn't used, as it can also be used to remove icons by simply not including them. (They also override each other) See reorder's notes for how to use this.
